THE DESIGN PROCESS

PLANNING




The planning stage is the most important, because the more precisely the project is planned, the less time it will take for the rest of the stages. What is decided and traced in this stage will define the rest of the project.

DESIGN




The design stage involves achieving the objectives outlined in the planning stage. Through prototypes and mockups, a visual representation is created where it is possible to predict how visitors will interact with it; it also helps to determine the user experience and if the visual style is meeting the determined objectives.

DEVELOPMENT




The development stage involves most of the programming work, consisting of the front-end and the back-end. The front-end consists of how visitors interact with the website when browsing it. The back-end is necessary to ensure that data is stored, manipulated, and delivered to the browser correctly.
